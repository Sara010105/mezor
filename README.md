
  # Kanban board with drag-and-drop

  This is a code bundle for Kanban board with drag-and-drop. The original project is available at https://www.figma.com/design/tG9Xy7gCq4AOkoD5Vr88Hn/Kanban-board-with-drag-and-drop.

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the Next.js app.

Run `npm run seed` to seed initial jewelry data.
